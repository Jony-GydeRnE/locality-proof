# §3 — Foundational lemma: $B = 0 \Rightarrow B_i = 0$

**Backs paper §3 (foundational lemma) and Item 1 of the master plan.**

The kill move (Step 1 of the mechanism) operates one homogeneity weight
at a time: it sends a chosen subset of free chords to $\infty$ and reads
off coefficients of the surviving Laurent polynomial. For this to be
valid term-by-term in $B$, we need that *if $B$ vanishes on the hidden
1-zero locus $\mathcal Z_r$, then each homogeneity weight-component
$B_i$ vanishes there separately*.

This generalizes Rodina's eqs ~15–17 from arXiv:2406.04234, which prove
the analogous statement for the tree amplitude. We prove it for the
fully general rational ansatz $B$.
