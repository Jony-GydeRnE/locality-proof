# §1 — Elementary geometric background

**Backs paper §1 (introduction) and Item 9 of the master plan
(undergraduate guide).**

This is the entry point for readers who have **never seen scattering
amplitudes before**. It builds up the polygon ↔ Feynman-diagram
dictionary from scratch, explains what a hidden zero is geometrically,
and gives a closed-timelike-curve heuristic for why locality has to
hold.

If you are reading this repo for the first time, **start here**.

## Files

- **`geometric_story.tex`** — source.
- **`geometric_story.pdf`** — compiled (the file to read).
- **`figures/`** — matplotlib-generated figures used in the document
  (regenerable from the scripts there).
- **`Intersections -> Violating Causality.HEIC`** — reference image for
  the causality argument.

The goal is that an undergraduate with one course in classical mechanics
can finish this document and understand what the rest of the repo is
trying to prove and why.
