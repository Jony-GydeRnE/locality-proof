# Independence of cyclic zones

**Script:** `independence_check.py`

## Question
The hidden-zero theorem uses all n cyclic 1-zero loci. How many of them are actually independent as constraints on the rational ansatz?

## Result
- n=5: only **3 of 5** loci are independent (legs 1, 2, 3 suffice).
- n=6: **5 of 6** are needed (corrects the floor(n/2) conjecture).

This is a new observation not present in Rodina's original paper.
