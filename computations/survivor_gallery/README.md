# Original hand-curated visualizations

These PDFs and images predate the parametric `step1_uncaught.py` script. They were produced by hand (Mathematica + manual layout) and show the non-local survivors at n=8 and n=9 as filled black polygons with red chord overlays.

The script-generated equivalents now live in `step1_uncaught/outputs/` and `step1_dual/outputs/`. These originals are kept for provenance.

## Files
- `nonlocal n=8.pdf` — original hand-curated n=8 survivor list.
- `nonlocal n=9.pdf` — original hand-curated n=9 survivor list.
- `IMG_8021.jpeg`, `unnamed (1).png`, `unnamed (2).png` — reference images (early Mathematica explorations).
